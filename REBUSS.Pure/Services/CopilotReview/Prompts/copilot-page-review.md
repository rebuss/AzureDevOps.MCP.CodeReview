Review the following code changes. For each issue found:
- Classify severity: critical | major | minor
- Specify the file path
- Describe the issue concisely
- Suggest a fix if applicable

Focus on: correctness, null safety, concurrency, async/await correctness, security, error handling, performance, missing tests.

Ignore minor style issues unless they affect correctness.

If no issues are found, state that explicitly.
